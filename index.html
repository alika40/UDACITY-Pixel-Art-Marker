  • Intro

    • Pixel art marker is a web application that offers a user an opporyunity to express themselves through painting.

  • Instructions On How To Paint

    1. To create a grid, enter a number into the respective input fields for grids' width and height, and click on "Create Grid" button.
    2. Click on the created grids to paint.
    3. Click on the color panel "GRID COLOR" to pick colors from the displayed color palette to change current color.
    4. Click on a grid to remove any existing applied color from the grid.
    5. Click on the color "BACKGROUND COLOR" to apply desired background color to the canvas and it can always be changed. But do ensure you pick a color and click "OK" to get the effect.
    6. NOTE: The maximum number of grid rows and columns that can be created is 50, while minimum is 5.
